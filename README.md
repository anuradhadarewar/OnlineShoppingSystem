# Online Shopping System

Java Console Based Online Shopping System Project.

## Features
- Login Authentication
- Product Management
- Add Product to Cart
- Remove Product from Cart
- Product Search
- Coupon Discount (SAVE10)
- Payment System (UPI, Card, Cash)

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- VS Code
- Git
- GitHub

## Project Structure
- Main.java
- Product.java
- Cart.java
- Customer.java
- Login.java
- Payment.java
